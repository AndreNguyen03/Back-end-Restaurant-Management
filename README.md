# Hướng dẫn cài đặt dự án

## Yêu cầu
- Node.js >= 16.x.x
- npm (đi kèm với Node.js)
- Git

## Cài đặt
1. Clone dự án:
   ```bash
   git clone <repository-url>
2. Di chuyển vào thư mục dự án:
   ```bash
   cd <tên-thư-mục-dự-án>
3. Cài đặt các package:
   ```bash
   npm install
   
## Chạy dự án
1. Chạy dự án:
   ```bash
   npm run dev
